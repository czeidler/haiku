Desktop Applets

Haiku fornisce alcuni strumenti utili che possono essere installati sul desktop o sulla deskbar attraverso i loro replicant.

LaunchBox  Tasti di scelta rapida per le tue applicazioni preferite.
NetworkStatus  La connessione è attiva?
PowerStatus  Mostra lo stato di carica della batteria del notebook.
ProcessController  Monitora e controlla ogni applicazione e servizio in esecuzione.
Workspaces  Una versione in miniatura di tutte le aree di lavoro.
