Preferenze

La filosofia chiave di Haiku è quella di ridurre le opzioni ed usare al loro posto impostazioni di default, ad ogni modo per preferenze specifiche, alcune cose devono essere comunque configurate o impostate. E' possibile trovare tutti i pannelli nel menu Preferenze della Deskbar.

Appearance Configura l'aspetto dell'interfaccia grafica.
Backgrounds Permette di scegliere un colore o un'immagine come sfondo per il Desktop o qualsiasi altra cartella.
DataTranslations Impostazioni per tutti i tipi di file supportati.
Deskbar Configura la Deskbar.
E-mail Configura gli account per le tue e-mail
FileTypes Aggiunge, rimuove e configura i tipi di file.
Keyboard Configura il ritardo di ripetizione e la frequenza.
Keymap Imposta la mappatura della tastiera.
Locale Configura il linguaggio del sistema.
Media Impostazioni audio e video, come ad esempio periferiche in/output e il mixer audio del sistema
Mouse Configura il mouse.
Network Configura la tua rete.
Printers Aggiunge, rimuove e configura le stampanti.
Screen Configura risoluzione, profondità, velocità di aggiornamento ed il numero di aree di lavoro utilizzate.
ScreenSaver Aggiunge, rimuove o configura uno screen saver.
Sounds Assegna suoni ai differenti eventi del sistema.
Time Configura l'ora, la data, e il fuso orario.
Tracker Configura il file manager di Haiku.
Touchpad Configura il touchpad.
VirtualMemory Imposta la quantità di spazio swap usato.
